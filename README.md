# Proyecto 1er

<h3>Descripcion</h3>
<p>este programa permite a los usuarios gestionar una lista de tareas que permite a los usuarios agregar, editar y eliminar tareas</p>

### Tecnologias utilizadas
<ul>
<li>HTML</li>
<li>CSS</li>
<li>JavaScript</li>
<li>Almacenamiento Local del Navegador</li>
</ul>
### Como usar la aplicacion

<p>Primero a ejecutar el programa nos aparece un recuadro que nos indica que coloquemos la tarea que deseamos realizar posteriormente a ingresar la tarea que deseamos nos mostrara la tarea que ingresamos en una después de lado derecho nos aparecerá una serie de botones que dice completar, editar y eliminar y en la parte superior donde nos muestra la tarea que ingresamos nos aparecen una serie de secciones que dicen todas, activadas y completadas si presionamos el botón de completar se subrayada la tarea que ingresa y si seleccionamos la sección de completadas se nos mostrara la tareas que completamos y si presionamos el botón editar se nos abrirá una ventana donde podremos cambiar el texto de la tarea y si presionamos el botón de eliminar se eliminara la tarea    </p>
